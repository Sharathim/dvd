DV Dream Homes AI Chatbot Integration

This guide explains how the DV Dream Homes chatbot is wired into the website.

Features: Groq API integration, conversation history, responsive UI, and a backend-protected API key flow.

Prerequisites

You need a Groq API key from:

https://console.groq.com/keys

Quick Start

Step 1: Add the stylesheet

In the <head> of each page:

<link rel="stylesheet" href="static/css/chatbot-widget.css">

Step 2: Add the widget script

Before the closing </body> tag:

<script src="static/js/chatbot-widget.js"></script>

Step 3: Configure Groq in the backend

Store the key in .env:

DV_GROQ_API_KEY=your-groq-api-key
DV_GROQ_MODEL=llama-3.1-8b-instant

The browser talks to /api/chatbot, and Flask forwards the request to Groq from the server side.

Current File Roles

  • static/css/chatbot-widget.css - Widget styling
  • static/js/chatbot-widget.js - Chat UI and frontend behavior
  • app.py - Backend chatbot route and Groq API call
  • .env - Groq model and API key configuration

Notes

  • The API key should stay in the backend only.
  • The model can be changed through DV_GROQ_MODEL without editing JavaScript.
  • The chatbot is already connected to the floating chat button used across the public pages.
