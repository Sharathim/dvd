🤖 DV Dream Homes AI Chatbot Widget Integration

This guide shows how to integrate the AI chatbot widget into your DV Dream Homes website.

✓ Features: Gemini API integration, conversation history, responsive design, matches your site design perfectly
---

📋 Prerequisites

You need a Google Gemini API key. Get one for free at:

https://makersuite.google.com/app/apikey

---

🚀 Quick Start (3 Steps)

Step 1: Add the CSS to your page

In the <head> section of your HTML file, add:

<link rel="stylesheet" href="static/css/chatbot-widget.css">

Step 2: Add the JavaScript to your page

At the end of the <body> section, add:

<script src="static/js/chatbot-widget.js"></script>

Step 3: Set your API Key

Open static/js/chatbot-widget.js and find this line (around line 135):

const API_KEY = 'YOUR_GEMINI_API_KEY';

Replace it with your actual Gemini API key:

const API_KEY = 'sk-1234567890abc...';

⚙️ Customization

Change the greeting message

In chatbot-widget.js, find the line in the createUI() method:

<div class="message-bubble bot">
  Hi! 👋 I'm the DV Dream Homes assistant. How can I help you today?
</div>

Edit the text to your preferred greeting.

Change the button position

In chatbot-widget.css, modify these values:

.chat-button-container {
  bottom: 28px;  /* Distance from bottom */
  right: 28px;   /* Distance from right */
}

Change the button icon

In chatbot-widget.js, find this line in createUI():

button.innerHTML = '💬';

Use any emoji or HTML you want. Examples: 🤖 💭 ❓ 🎯

Adjust panel size

In chatbot-widget.css, modify:

.chat-panel {
  max-width: 420px;  /* Width on desktop */
  height: 600px;     /* Height */
}

🐛 Troubleshooting

Chatbot not appearing?

  • Check that both files are linked correctly in your HTML
  • Check browser console for errors (F12 → Console tab)
  • Make sure your API key is set

API key error?

  • Check you have a valid Gemini API key from Google
  • Check there are no extra spaces in your API key
  • Verify the key has appropriate permissions

Messages not sending?

  • Check your API key is correct
  • Check your internet connection
  • Open developer console (F12) to see specific error messages
